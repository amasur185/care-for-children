# Care For Children — Adhya Masur's Fundraiser

A personal fundraising website in support of [Care For Children](https://www.careforchildren.org/),
which provides free holistic education to underprivileged children across rural India.

Donations are processed directly by Care For Children via their official platform:
**https://careforchildren.kindful.com/**

## Structure

- `index.html` — the single-page site
- `styles.css` — styling

## Adding a photo

Save your picture as `photo-placeholder.jpg` in this folder and it will appear in the hero
section automatically.

## Hosting

This is a static site (no build step) and is deployed via:

- **GitHub Pages** — served from the repository root
- **Netlify** — connected to this repo for continuous deploys (`netlify.toml` included)
